Redesign

New Hampshire Unemployment Website

Project Overview

The current website suffers from numerous accessibility issues, which have resulted in a high bounce rate as evidenced by website analytics. To address these challenges, the website redesign project seeks to create a user-friendly interface that enables visitors to easily navigate the website. The primary goal is to increase user engagement by encouraging visitors to complete tasks and browse the website beyond their initial interests.

My Role

As a member of a four-person team, my primary responsibility is to collaborate with other UX and UI designers throughout the entire website redesign process.

This include:

  • conducting primary user research to gain insights into user needs and preferences
  • prioritizing problems to be addressed
  • creating wireframes to visualize design concepts
  • performing A/B testing to validate design decisions

Together with the team, my aim is to deliver an optimized user experience that meets the needs of the target audience and achieves the project goals.
